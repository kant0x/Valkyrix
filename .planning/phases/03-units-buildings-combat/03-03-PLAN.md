    placeBuilding(type, tileCol, tileRow, zoneLayer, mapWidth, state):
    - Returns false if zone tile is empty (zoneLayer[row*mapWidth+col] === 0)
    - Returns false if another building already occupies the same (tileCol, tileRow)
    - Returns false if state.resources < cost (attack=50, buff=40)
    - On success: deducts cost from state.resources, adds Building to state.buildings, returns true
    - Created building has correct default attackCooldown (0), correct radius, damage, attackRate, resourceRate, buffValue per type

    sellBuilding(buildingId, state):
    - Removes building from state.buildings by id
    - Adds 60% of cost back to state.resources (attack→+30, buff→+24)
    - Returns true; returns false if id not found

    update(dt, state) — attack towers:
    - Tower with attackCooldown > 0: decrements by dt, no action
    - Tower with attackCooldown <= 0 and enemy within radius: spawns Projectile in state.projectiles, resets attackCooldown to 1/attackRate

    - Dead projectiles (no target or hit) filtered from state.projectiles each frame

    ResourceSystem.update(dt, state):
    - Each building generates resourceRate electrolatov/s: state.resources += building.resourceRate * dt for each building
    - No kill-drop logic here (CombatSystem calls registerKill — handled in Plan 04 CombatSystem)
