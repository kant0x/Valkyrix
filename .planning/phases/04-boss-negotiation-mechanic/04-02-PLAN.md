Detection (BOSS-01):
- update(dt, state, container) detects boss: state.units.find(u => u.def.role === 'boss' && u.faction === 'enemy' && u.hp > 0 && !u.def.enraged)
- If boss found AND bossNegotiation not yet triggered → set state.phase = 'negotiation', set state.bossNegotiation = { active: true, triggered: true }, mount overlay with callbacks
- If state.bossNegotiation?.triggered is true → skip (never re-trigger)
- If boss is enraged (u.def.enraged === true) → skip
- If state.phase !== 'playing' → skip (negotiation already in progress or game ended)

Success path (BOSS-03):
- handleSuccess(state): filter boss from state.units, state.resources += 120, state.citadelHp = Math.min(state.citadelMaxHp, state.citadelHp + 400), state.waveTimer = Math.max(state.waveTimer, 20), state.phase = 'playing', state.bossNegotiation = { active: false, triggered: true, outcome: 'success' }, cleanup overlay

Failure path (BOSS-04):
- handleFailure(state, boss): boss.def = { ...boss.def, enraged: true, damage: Math.round(boss.def.damage * 1.5) }, enqueueHorde(state), state.phase = 'playing', state.bossNegotiation = { active: false, triggered: true, outcome: 'failure' }, cleanup overlay
- enqueueHorde: push { 'light-enemy': 12, 'heavy-enemy': 6, 'ranged-enemy': 4 } at powerScale 1.4, delay increments of 0.4s starting at 1.0s

forceReset (cleanup on unmount):
- If overlay exists, call unmount(); set negotiationActive = false; if state.phase === 'negotiation', set state.phase = 'playing'

```typescript
import type { GameState, Unit } from './game.types';
import { NegotiationOverlay } from '../screens/NegotiationOverlay';

export const NEGOTIATION_RESOURCE_REWARD = 120;
export const NEGOTIATION_HP_REWARD = 400;
export const NEGOTIATION_WAVE_TIMER_FLOOR = 20;
export const HORDE_POWER_SCALE = 1.4;

export class BossSystem {
  private overlay: NegotiationOverlay | null = null;

  update(dt: number, state: GameState, container: HTMLElement | null): void {
    // Guards: skip if not playing or negotiation already triggered
    if (state.phase !== 'playing') return;
    if (state.bossNegotiation?.triggered) return;

    const boss = state.units.find(
      u => u.def.role === 'boss' && u.faction === 'enemy' && u.hp > 0 && !u.def.enraged
    );
    if (!boss) return;

    // Freeze boss in place (set state before mounting overlay)
    boss.state = 'fighting';
    boss.fightingWith = null;
    state.phase = 'negotiation';
    state.bossNegotiation = { active: true, triggered: true };

    if (container) {
      this.overlay = new NegotiationOverlay();
      this.overlay.mount(container, {
        onSuccess: () => this.handleSuccess(state),
        onFailure: () => this.handleFailure(state, boss),
      });
    }
  }

  handleSuccess(state: GameState): void { ... }
  handleFailure(state: GameState, boss: Unit): void { ... }
  private enqueueHorde(state: GameState): void { ... }
  forceReset(state: GameState): void { ... }
  private cleanup(): void { this.overlay?.unmount(); this.overlay = null; }
}
```

For unit tests, pass `container = null` so NegotiationOverlay is never instantiated. Call handleSuccess/handleFailure directly with a mock GameState to test outcomes in isolation. This avoids jsdom dependency in BossSystem tests.
