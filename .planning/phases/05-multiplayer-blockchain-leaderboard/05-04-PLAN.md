AnchorProvider pattern for browser wallet:
```typescript
const anchorWallet = {
  publicKey: new PublicKey(walletState.publicKey),
  signTransaction: <T>(tx: T) => provider.signTransaction!(tx),
  signAllTransactions: <T>(txs: T[]) => provider.signAllTransactions ? provider.signAllTransactions(txs) : Promise.resolve(txs),
};
const anchorProvider = new AnchorProvider(connection, anchorWallet as never, { commitment: 'confirmed' });
```

PITFALL: `AnchorProvider` publicKey must be `new PublicKey(string)`, NOT raw string.
PITFALL: SOAR game account requires one-time `initializeNewGame()` — NOT called at runtime. Store PDA as constant.
