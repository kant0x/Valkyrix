**BLOCKER: anchor build падает из-за Solana 1.18 + Cargo.lock v4 несовместимости**

- Solana 1.18 бандлит cargo 1.75.0
- Зависимости Anchor 0.32.1 тянут `toml_datetime v1.1.0+spec-1.1.0` с `edition2024`
- edition2024 требует cargo >= 1.85
- Попытки: Rust 1.79 (слишком старый), патч Cargo.lock v4→v3 (не помогло), SBF_SDK_PATH (не помогло)


</blockers>
